### NGINX

- `NGINX_HOST` - адрес на котором будет доступно приложение;
- `NGINX_PORT` - порт на котором будет доступно приложение.

### MYSQL

- `MYSQL_HOST` - хост на котором будет доступен сервис mysql;
- `MYSQL_PORT` - порт на котором будет доступен сервис mysql;
- `MYSQL_ROOT_PASSWORD` - пароль root пользователя mysql.

### REDIS

- `REDIS_HOST` - хост на котором будет доступен сервис redis;
- `REDIS_PORT` - порт на котором будет доступен сервис redis.
